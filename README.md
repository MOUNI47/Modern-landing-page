<h1>🚀 Modern Product Landing Page</h1>
<h2>📌 Project Description</h2>
<p>
This project is a modern and responsive Product Landing Page built using 
<strong>React</strong> with proper component structure, UI hierarchy, 
reusable components, and clean spacing.
</p>
<p>
The application demonstrates strong frontend development practices including 
responsive design, hover effects, and component-based architecture.
</p>
<h2>✨ Features</h2>
<ul>
<li>Hero Section with heading and Call-to-Action button</li>
<li>Reusable Feature Card components (minimum 3 cards)</li>
<li>Hover effects on buttons and cards</li>
<li>Proper alignment and spacing</li>
<li>Fully responsive for mobile and desktop</li>
</ul>
<h2>🛠️ Technologies Used</h2>
        <ul>
            <li>React (Vite)</li>
            <li>Functional Components</li>
            <li>CSS for styling</li>
            <li>Responsive Design using Media Queries</li>
        </ul>
<h2>▶️ How to Run the Project</h2>
<p>Follow these steps:</p>
        <ol>
            <li>Install dependencies: <code>npm install</code></li>
            <li>Start development server: <code>npm run dev</code></li>
            <li>Open browser and visit: <code>http://localhost:5173</code></li>
        </ol>
<h2>🎯 Learning Objectives</h2>
        <ul>
            <li>Understand component-based architecture in React</li>
            <li>Create reusable components</li>
            <li>Apply responsive CSS styling</li>
            <li>Implement hover effects and transitions</li>
        </ul>
